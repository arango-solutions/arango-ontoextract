# Demo Sample Documents

These Markdown files are synthetic and safe to upload for AOE demonstrations.
They are written with explicit headings because the document chunker uses
structure to produce better extraction units.

## Larger Demonstration Documents

Use these when you want richer extraction runs with many classes, relations,
events, controls, metrics, and workflows:

| File | Demonstration Focus |
| --- | --- |
| `global-bank-financial-crime-investigation.md` | AML investigation, customers, accounts, transactions, alerts, cases, evidence, regulatory reporting |
| `integrated-healthcare-delivery-network.md` | Healthcare delivery operations, encounters, orders, results, medication management, care gaps, claims, research |
| `global-supply-chain-resilience-program.md` | Supplier networks, BOMs, procurement, transportation, inventory, quality, disruption response |
| `energy-utility-grid-operations.md` | Electric utility assets, feeders, meters, outages, work orders, DERs, incident response, reliability metrics |
